# Henshusha Edit Timeline

Use when the user wants to improve subtitles, pick a cut, create a short, or prepare a timeline for render.

## Workflow

1. Read the target `timelines/*.timeline.json`.
2. Clean transcript text without losing meaning.
3. Split subtitle cues by timing, reading speed, and line length.
4. Mark important words for highlight when useful.
5. Choose or update render variant: 16:9, 9:16, or 1:1.
6. Apply an art direction preset.
7. Save the updated timeline and summarize changes.
