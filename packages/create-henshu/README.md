# create-henshu

Create a Henshusha video editing workspace.

```bash
bun create henshu my-video-project
# or
npm create henshu my-video-project
pnpm create henshu my-video-project
```

The generated workspace includes:

- `.claude/skills/`
- `.codex/skills/`
- `.pi/skills/`
- `sources/raw/`
- `transcripts/`
- `timelines/`
- `renders/`
- sample Timeline JSON

Henshusha is early. The first package focuses on creating the agent-editable workspace shape.
